# Proceso de Certificación

El proceso de certificación es un paso obligatorio ante la DGII para que tu empresa pueda emitir comprobantes fiscales electrónicos (e-CF) en producción. A continuación, se detallan los datos y configuraciones necesarias para completar este proceso.

## Datos del software a utilizar

Al momento de registrar tu software en el portal de la DGII para el proceso de certificación, deberás completar los siguientes campos:

| Campo                    | Valor   |
| ------------------------ | ------- |
| **Tipo de software**     | Externo |
| **Nombre del software**  | MSeller |
| **Versión del software** | 1.0     |

## URLs de integración

Las URLs que debes configurar en el formulario de certificación son:

| Campo                           | URL                                   |
| ------------------------------- | ------------------------------------- |
| **URL de recepción**            | `https://ecf.api.mseller.app/CerteCF` |
| **URL de aprobación comercial** | `https://ecf.api.mseller.app/CerteCF` |
| **URL de autenticación**        | _(Dejar en blanco)_                   |

> **Nota**: La URL de autenticación se deja en blanco porque MSeller ECF maneja la autenticación internamente a través de su propia API.

## Datos del proveedor

Los datos del proveedor que debes ingresar en el formulario de certificación son:

| Campo                   | Valor       |
| ----------------------- | ----------- |
| **RNC/Cédula**          | 130862346   |
| **Nombre/Razón social** | ITSoluclick |
| **Nombre comercial**    | MSeller     |

## Pasos para completar la certificación

1. Accede al portal de la DGII para el proceso de certificación de e-CF
2. Completa el formulario con los **datos del software** indicados arriba
3. Ingresa las **URLs de integración** tal como se especifican
4. Rellena los **datos del proveedor** con la información proporcionada
5. Haz clic en **"Generar Archivo"** para generar el archivo de configuración
6. Asegúrate de tener una API Key creada para el entorno **CerteCF** en [https://ecf.mseller.app/api-keys/list](https://ecf.mseller.app/api-keys/list)

> **Importante**: Antes de iniciar el proceso de certificación, asegúrate de haber completado las pruebas en el entorno **TesteCF** satisfactoriamente.
