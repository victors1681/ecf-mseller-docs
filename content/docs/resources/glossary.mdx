# Glosario de Términos

## A

**API (Application Programming Interface)**: Conjunto de reglas y protocolos que permite a diferentes aplicaciones comunicarse entre sí.

**API Key**: Clave única que identifica y autoriza a un sistema o aplicación para usar un servicio o API.

## C

**Certificado Digital**: Documento electrónico emitido por una autoridad certificadora que garantiza la identidad del titular y permite firmar digitalmente documentos.

**CertCF**: Entorno de certificación en la plataforma MSeller ECF, utilizado durante el proceso de certificación ante la DGII.

## D

**DGII**: Dirección General de Impuestos Internos, entidad gubernamental de República Dominicana encargada de la administración y recaudación de los principales impuestos internos y tasas.

**Documento XML**: Formato de documento estructurado utilizado para el intercambio de datos que sigue el estándar XML (eXtensible Markup Language).

## E

**e-CF**: Comprobante Fiscal Electrónico, documento digital que sustituye las facturas tradicionales en papel, con el mismo valor legal.

**eCF**: Entorno de producción en la plataforma MSeller ECF, utilizado para la emisión de facturas electrónicas oficiales.

**Emisor Electrónico**: Contribuyente autorizado por la DGII para emitir comprobantes fiscales electrónicos.

## F

**Factura Electrónica**: Documento digital que cumple con los requisitos legales y reglamentarios exigibles a las facturas tradicionales, garantizando la autenticidad de su origen y la integridad de su contenido.

**Firma Digital**: Mecanismo criptográfico que permite verificar la autenticidad e integridad de un mensaje o documento digital.

## H

**Hash**: Función matemática que genera una cadena de caracteres de longitud fija a partir de un documento, utilizada para verificar la integridad de los datos.

## P

**Plan de Contingencia**: Conjunto de procedimientos alternativos para garantizar la continuidad de la emisión de comprobantes fiscales en caso de fallas en el sistema principal.

## R

**Receptor Electrónico**: Contribuyente que recibe comprobantes fiscales electrónicos y puede validarlos digitalmente.

**RNC (Registro Nacional del Contribuyente)**: Número de identificación fiscal asignado por la DGII a personas jurídicas y negocios de único dueño en República Dominicana.

## T

**TesteCF**: Entorno de pruebas en la plataforma MSeller ECF, utilizado durante el desarrollo e integración inicial.

**Token**: Cadena de caracteres que permite la autenticación temporal en una API o servicio, generalmente con un tiempo de validez limitado.

## X

**XML (eXtensible Markup Language)**: Lenguaje de marcado que define un conjunto de reglas para la codificación de documentos en un formato legible tanto para humanos como para máquinas.

**XSD (XML Schema Definition)**: Lenguaje de esquema utilizado para describir la estructura y las restricciones de contenido de documentos XML.
